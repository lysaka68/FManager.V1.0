
# 💰 Финансов Мениджър (Financial Manager)

Едно просто и интуитивно приложение за управление на личните финанси, създадено с **Electron**.

---

## 🇧🇬 Описание на български

Това приложение ви помага да следите своите приходи и разходи с лекота. Създадено е да бъде леко, бързо и лесно за употреба.

### 🚀 Как да стартирате програмата:
Тъй като приложението е преносимо (portable), не е нужна инсталация. Следвайте тези стъпки:
1. Отидете в секцията [Releases](https://github.com/lysaka68/FManager.V1.0/releases/tag/v1.0) и изтеглете най-новия `.zip` архив.
2. **Важно:** Разархивирайте целия архив в папка по ваш избор.
3. Стартирайте файла `Финансов Мениджър.exe`.

> [!NOTE]
> Не премествайте `.exe` файла извън неговата папка, тъй като той зависи от придружаващите го библиотеки (като `ffmpeg.dll`).

---

## 🇺🇸 Description in English

A simple and intuitive personal finance management application built with **Electron**.

### 🚀 How to run:
The application is portable, so no installation is required. Follow these steps:
1. Go to the [Releases](link-to-your-release-here) section and download the latest `.zip` archive.
2. **Important:** Extract the entire ZIP archive into a folder of your choice.
3. Run the `Финансов Мениджър.exe` file.

> [!CAUTION]
> Do not move the `.exe` file out of its folder, as it relies on the accompanying library files to function correctly.

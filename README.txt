# 💰 Личен Финансов Мениджър — Десктоп Приложение

## ⚡ Бързо стартиране (3 стъпки)

---

### Стъпка 1 — Инсталирайте Node.js (еднократно)

1. Отидете на https://nodejs.org
2. Свалете версията **"LTS"** (зеленият бутон)
3. Инсталирайте го с Next → Next → Finish

---

### Стъпка 2 — Инсталирайте Electron

1. Отворете **Command Prompt** (Старт → напишете `cmd` → Enter)
2. Навигирайте до папката с приложението:
   ```
   cd C:\Users\ВАШЕТО_ИМЕ\Downloads\finance-app
   ```
3. Изпълнете:
   ```
   npm install
   ```
   *(изчакайте — сваля ~100MB, само веднъж)*

---

### Стъпка 3 — Стартирайте

```
npm start
```

Приложението ще се отвори в собствен прозорец — **без браузър!** 🎉

---

## 🖥️ Създаване на .EXE инсталатор (по желание)

Ако искате да създадете истински `.exe` файл за инсталация:

```
npm run build-win
```

Готовият инсталатор ще се появи в папка `dist/`.

---

## ❓ Чести проблеми

**"npm не се разпознава"** → Node.js не е инсталиран правилно. Рестартирайте компютъра след инсталацията.

**"Cannot find module 'electron'"** → Изпълнете отново `npm install` в правилната папка.

**Приложението не се отваря** → Уверете се, че сте в папката `finance-app` преди `npm start`.
